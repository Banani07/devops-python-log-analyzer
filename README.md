# 🚀 DevOps Python Log Analyzer

## 📌 Overview
This project is a simple yet practical **log analysis tool** built using Python.  
It helps in parsing application logs and identifying the number of ERROR and INFO messages, which is useful for basic monitoring and debugging in DevOps workflows.

---

## 🧠 Problem Statement
In real-world systems, logs are generated continuously. Manually analyzing logs to identify issues is time-consuming.

This project automates:
- Log parsing  
- Error detection  
- Basic metrics generation  

---

## ⚙️ Features
- Reads log files line by line  
- Counts ERROR and INFO occurrences  
- Calculates total log entries  
- Simple and lightweight implementation
